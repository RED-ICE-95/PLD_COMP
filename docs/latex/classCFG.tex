\doxysubsection{Description détaillée}
Graphe de contrôle de flux et table des symboles associée. 

The class for the control flow graph, also includes the symbol table

Le \mbox{\hyperlink{classCFG}{CFG}} gère les blocs de base, la génération d\textquotesingle{}assemblage et la résolution des variables temporaires et locales pour une fonction donnée.
