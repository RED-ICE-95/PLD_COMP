\doxysubsection{Description détaillée}
Visiteur de génération de code pour l\textquotesingle{}IR et la \mbox{\hyperlink{classCFG}{CFG}}. 

\mbox{\hyperlink{classCodeGenVisitor}{Code\+Gen\+Visitor}} visite les nœuds du parse tree et déclenche l\textquotesingle{}émission des instructions intermédiaires et des blocs de contrôle de flux.
