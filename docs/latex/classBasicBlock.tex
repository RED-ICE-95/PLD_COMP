\doxysubsection{Description détaillée}
Bloc basique dans le graphe de contrôle de flux. 

The class for a basic block

Un \mbox{\hyperlink{classBasicBlock}{Basic\+Block}} contient une liste d\textquotesingle{}instructions IR et des connexions vers ses successeurs vrai/faux pour la génération de sauts conditionnels.
